# 🌍 Impact Horizon – ONG pour l'autonomisation des jeunes et des femmes

## 📋 Description du projet

**Impact Horizon** est une ONG béninoise engagée dans le développement socio-économique durable. Le site web présente les services, programmes et actions menés par l'organisation pour favoriser l'autonomisation et l'insertion professionnelle des jeunes (14-35 ans) et des femmes au Bénin.

**Zones d'intervention** : Cotonou & Abomey-Calavi

---

## 🎯 Objectif

Faciliter l'accès des jeunes et des femmes à des compétences pratiques et entrepreneuriales pour favoriser :
- L'insertion professionnelle
- L'autonomie économique
- Le développement durable

## 💡 Mission & Vision

### Mission
Contribuer au développement socio-économique durable en facilitant l'accès à des compétences pratiques et entrepreneuriales.

### Vision
Une société où jeunes et femmes deviennent acteurs du développement.

### Valeurs fondamentales
- ✅ Engagement social
- ✅ Autonomisation
- ✅ Innovation
- ✅ Solidarité
- ✅ Inclusion
- ✅ Développement durable

---

## 📚 Programmes & Activités

### 1. **Formation professionnelle**
- Peinture bâtiment
- Électricité
- Maintenance
- Communication
- Métiers techniques

### 2. **Ateliers & séminaires**
Sessions pratiques et intensives pour renforcer les compétences

### 3. **Accompagnement entrepreneurial**
Appui à la création et au développement de micro-entreprises pour les jeunes

### 4. **Autonomisation des femmes**
- Activités génératrices de revenus (AGR)
- Mentorat
- Microcrédit

### 5. **CAMP MÉTIERS-VACANCES (CMV)**
Pendant les vacances, les jeunes découvrent et apprennent différents métiers

### 6. **Insertion & sensibilisation**
- Accompagnement vers l'emploi
- Actions communautaires pour le développement durable

## 🚀 Installation & utilisation

### 1. **Cloner ou télécharger le projet**
```bash
git clone <url-du-repo>
cd site_impact_horizon
```

### 2. **Ouvrir dans un navigateur**
- Double-cliquez sur `index.html`, ou
- Ouvrez depuis VS Code avec **Live Server** :
  - Clic droit sur `index.html` → **Open with Live Server**

### 3. **Déploiement**
Le site est prêt pour être hébergé sur tout serveur web :
- Hosting statique (Netlify, GitHub Pages, Vercel, etc.)
- Serveur HTTP classique
- OVH, AWS, Azure, etc.
